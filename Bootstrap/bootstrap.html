Bootstrap

Man kann Bootstrap von deren Website herunterladen oder die cdn also die Links von Bootstraps Webseite verlinken lassen. Wenn man Bootstrap installiert hat, kann man sie in einem Ordner verlegen und dann die Source (Wo es auf dem PC ist) eingeben. Alles muss im head eingegeben werden.
Man kann das Grid beliebig einstellen, aber es hat ein Limit von 12 Spalten und innerhalb dieser Spaltenanzahl kann man es wie gewünscht einstellen. Wenn es grösser wird kann man den Grid entweder kleiner oder grösser einstellen und es geht auch umgekehrt, wenn man es kleiner stellt. Man bestimmt nun über CSS Klassen wie viele Spalten in einer Zeile angezeigt werden und ab welcher Gerätegröße dies passieren soll.
Bootstrap gibt schon ein paar Sachen vor, was Design angeht, also könnte man theoretisch dort Copy&Pasten. Sonst kann man alles durch herausfinden ein bisschen einstellen oder man kann alle CSS-Dateien herunterladen. Dort kann man dann durch reinschauen herausfinden, was man alles nutzen kann und wie man es macht. Durch das reinschauen und inspizieren der Dateien, wird der Rest einfach und man kann alles implementieren. Zum Beispiel beim Karussel nutzt man Slide und Nummern, um die Slide Nummer anzugeben und die wie vielte Slide wie erscheinen soll. Es gibt mehrere IDs und Klassen, die man nutzen kann, um diese Slide zu designen. Ich habe das CSS kopiert und dann versucht die Slide selbst einzurichten mit Hilfe der Webseite von Bootstrap.
Man kann sich sehr viel Hilfe von den Webseiten von Bootstrap holen.
